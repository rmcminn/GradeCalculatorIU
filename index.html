IU GPA Calculator

A GPA calculator for students of Indiana University - Bloomington

Download this project as a .zip file Download this project as a tar.gz file

Download - Latest Release

How To Install

Once you have the .jar downloaded, make sure you have the latest version of Java RunTime installed, then just double click on the .jar file, and you're ready to go.

Why?

I'm the kind of guy that likes to know where my GPA stands for a current semester, and I found Indiana University's web tool really tedious. So, I developed a desktop application that I could just run fast and easy.
